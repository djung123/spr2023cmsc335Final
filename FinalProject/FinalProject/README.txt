App Description:
    This application is similar to Piazza, where
    you can read, post, modify posts. Only members can write or read posts.
    Keeps the track of the session for users.

API Used: 
    This application is using 2 APIs:
    Auth0 API: Authentication API
    Link: https://auth0.com/docs/api
    HTML Sound(Audio) API
